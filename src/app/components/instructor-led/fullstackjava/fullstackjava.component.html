Full Stack Java Development- Instructor led on-line Training
This will help an engineer understand end-to-end application development with Java and related technologies. This course will focus on design and architecture of application, build user-facing website, create and maintain database to support the application and test and maintain application from a full-stack java engineer point of view.
Technologies:
Java, Java EE, JSP, Servlets, Eclipse, Maven, Spring, Spring boot, Spring MVC, Spring jdbctemplate, JUnit, JPA, RESTful API, Angular, MySQL

Introduction to Java Programming and Algorithms in Java:
Lessons: Java and Advance Java and Algorithms (Sorting, List, Queue, Recursion)
RDBMS design Principles and MySQL
Lessons: Concepts of RDBMS, Database design principles, Query and MySQL
Java Web Technologies with Servlet, JSP and JSTL:
Lessons: Java EE, Web technology, Servlet API, JSP, Tags. Learn with a project.
Java Web Development with Maven, MySQL:
Lessons: Continue with the project with Maven build and backend MySQL database with JDBC.
Java Web Application development with Spring MVC, JPA, and testing with Spring Junit:
Lessons: Continue with the project and include Spring MVC, Java Persistence API and introduction to Testing with Junit.
RESTful Application Development with Java, Spring Boots, Spring JdbcTemplate, Postman and Angular. Part- 1
Lessons: Java EE, Web technology, Servlet API, JSP, Tags. Learn with a project.
RESTful Application Development with Java, Spring Boots, Spring JdbcTemplate, Postman and Angular. Part -2:
Lessons: Introduction to Angular and connect with the ongoing project. Use of Postman for validating RESTful APIs.
Java Web Application development with Spring MVC, JPA, and testing with Spring Junit:
Lessons: Continue with the project and include Spring MVC, Java Persistence API and introduction to Testing with Junit.
Automation
Concepts of Continues Integration (CI) and Continuous Delivery (CD) and building a pipeline. Concepts of Docker
